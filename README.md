# HomeEstate - Real Estate Platform

HomeEstate is a modern and secure real estate platform in Latvia, designed to simplify the process of renting and buying properties. The website features a beautiful, responsive UI, an advanced search system, and a dedicated owner panel for managing property listings with Stripe-powered payment integrations for premium listing plans.

**Visit the website:** [http://homeest.unaux.com/](http://homeest.unaux.com/)

---

### **Homepage**
![Homepage](Images/Homepage.png)
The homepage serves as the entry point for users. It features a modern hero section with a quick search bar for city, type, and price range. It also showcases the latest active property listings directly from the database, along with platform statistics.

---

### **Par Mums**
![Par Mums](Images/ParMums.png)
The "About Us" page explains the mission and vision of HomeEstate. It uses interactive animations and decorative shapes to tell the story of the platform, highlighting its goal to make the Latvian real estate market more transparent and accessible.

---

### **Meklēt īpašumu**
![Meklēt īpašumu](Images/MekletIpasumu.png)
This section allows users to browse all active listings. It includes a comprehensive filtering system to find properties by location, type (buy/rent), price range, and category (apartment, house, land). The results are displayed in a modern grid layout with quick-view cards.

---

### **Izveidot Sludinājumu**
![Izveidot Sludinājumu](Images/IzveidotSludinājumu.png)
A dedicated multi-step form for property owners to create new listings. This feature-rich interface guides users through entering basic info, descriptions, amenities, and uploading high-quality images. It includes real-time validation to ensure complete and accurate property data.

---

### **Kļūsti par īpašnieku**
![Kļūsti par īpašnieku](Images/KlustiParIpasnieku.png)
This page highlights the benefits of the "Pay-to-List" model. It explains the different subscription plans (Silver and Gold) available for owners, integrated with Stripe for secure and easy payments, allowing them to gain more visibility for their properties.

---

### **Mani Sludinājumi**
![Mani Sludinājumi](Images/ManiSludinajumi.png)
The owner's personal dashboard where they can manage all their property listings. It provides a clear overview of active, draft, and expired listings, allowing for quick edits, deletions, or publishing of drafts.

---

### **Pierakstīšanās**
![Pierakstīšanās](Images/Pierakstīšanās.png)
A secure and user-friendly login page. It provides access for both regular users and property owners, ensuring a personalized experience across the platform while keeping user data protected.

---

### **Reģistrēties**
![Reģistrēties](Images/Registreties.png)
The registration page where new users can join the platform. It allows users to select their role (User or Owner) during sign-up, which dynamically changes the available features after account creation.

---

### **Administrācijas Panelis**
![Administrācijas Panelis](Images/Administracijas.png)
A powerful administrative backend for site moderators and admins. It allows for full management of users, property listings, and platform-wide settings, ensuring the quality and safety of the content on HomeEstate.

---

### **Datu Tabula**
![Datu Tabula](Images/Tabula.png)
A detailed data management view within the admin panel. It provides a spreadsheet-like interface for advanced filtering, sorting, and bulk management of the platform's database, including user roles and listing statuses.
